ML Visualizer

This visualizer allows you to generate random data points and run machine learning algorithms to classify a target point. You can move points around to see how the classifications change in real-time.

About

This project offers an interactive platform to explore three fundamental machine learning algorithms: K-Nearest Neighbors (KNN), Support Vector Machines (SVM) and Linear Regression (LR). These algorithms are crucial in supervised learning, where the goal is to classify data points based on labeled training data.

K-Nearest Neighbors (KNN)

KNN is a non-parametric algorithm used for classification and regression tasks. It works by finding the 'K' nearest data points in the training set and determining the majority class among them to classify a new data point. This method is particularly intuitive and effective for small datasets.

Example: Consider a scenario where you want to classify a fruit as either an apple or an orange. By looking at the closest fruits in terms of features like color and size, KNN can determine the fruit's category.

Figure 1: Example of KNN classification. The red dot's classification is determined by the majority of its nearest neighbors.

Support Vector Machines (SVM)

SVM is a powerful classification algorithm that finds the hyperplane which best separates different classes in the dataset. The algorithm aims to maximize the margin between the classes, thus providing a robust decision boundary. SVM can handle linear and non-linear classification tasks by using kernel functions to transform data into higher dimensions.

Example: For classifying emails as spam or not spam, SVM can be used to draw an optimal boundary based on the features of the emails, like word frequency and presence of certain phrases.

Figure 2: Example of SVM classification. The solid line represents the optimal hyperplane, and the dashed lines represent the margins.

Linear Regression (LR)

Linear Regression is a fundamental algorithm used for predicting a continuous target variable based on one or more input features. It finds the best-fitting line through the data points by minimizing the sum of the squared differences between the observed and predicted values.

Example: Consider predicting the price of a house based on its size. Linear Regression can fit a line to the data points representing house prices and sizes, allowing you to predict the price of a new house based on its size.

Figure 3: Example of Linear Regression. The line represents the best fit through the data points.

How to Use the Visualizer

Follow these steps to explore the algorithms:

  • Generate data points by clicking 'Generate Data'. Adjust the number of points, categories, and cluster percentage as needed.
  • Select the model you want to visualize: KNN or SVM.
  • For KNN, specify the number of nearest neighbors (K) and weighting method.
  • For SVM, simply click 'Run SVM' to see the optimal hyperplane.
  • For LR, choose the error view option and click 'Run Linear Regression' to see the best-fit line.
  • Drag data points around the canvas to see how the classification and decision boundaries change in real-time.
