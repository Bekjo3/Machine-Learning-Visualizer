# Machine Learning Visualizer

**[Live Demo](https://ml-visualizer.vercel.app/)**

An interactive web app that visualizes how fundamental machine learning algorithms operate. You can generate random datasets, adjust algorithm parameters, and drag data points around the canvas to see classifications and decision boundaries update in real-time.

## Features

* **K-Nearest Neighbors (KNN):** Adjust the number of neighbors (K) and weighting methods to see how majority-class predictions shift.
* **Support Vector Machines (SVM):** Visualize the optimal hyperplane and margins separating different data classes.
* **Linear Regression (LR):** View the best-fit line and observe how minimizing squared differences works interactively.
* **Interactivity:** Drag and drop data points on the canvas to see the impact on the active model.
